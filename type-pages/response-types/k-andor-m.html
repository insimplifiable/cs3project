PC

Two major input devices of a computer are the keyboard, where it receives input by pressing keys, and the mouse, a device that moves the on-screen cursor or any of its variations according to hand movements. With these, they make the input devices for this type.

When using a keyboard as an input device, keybinds, or specific keys for you to press, are usually preset and can be changed, such that keys pressed indicate game input on some certain place. Some even use a keypad with a set number of keys to match with the number you need in game.

And, when using a mouse as an input device, they usually indicate that there is a mix of aim used. Sometimes if compatible and really needed, tablet is used instead of mouse to primarily alleviate mouse drift and to have better control as similar to handwriting.
