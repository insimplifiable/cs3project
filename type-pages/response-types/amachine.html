Arcade Machine

Arcade machine games are basically ones that are generally seen in arcades. Although very rare for players to do so, one can possess such through payment.

Just like portable controllers, it requires a new set of skills to be able to master each game, except if some bear huge similarities with the conventional input devices. The difference between them is in their name: portability, as you don't just bring them anywhere for fun.
