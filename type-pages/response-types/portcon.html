Portable Controller

This type includes the use of, of course, a portable controller. Controllers are generally defined in this genre as a device with a different way of input compared to the common touchscreen, keyboard, and mouse.

Like arcade machines, they are very unique due to how much potential a game can be by creating a different controller with unconventional inputs and hence implies learning another set of skills to master.
